# Linked Packages

You can use these packages in any standalone project by running the following commands in your standalone project's terminal:

```bash
bun link @radiant/cli
bun link vscode-radiant
bun link @codesordinatestudio/radiant-bun
bun link @codesordinatestudio/radiant-plugin-durable-streams
bun link @codesordinatestudio/radiant-plugin-s3
bun link @codesordinatestudio/radiant-plugin-surrealdb
bun link @codesordinatestudio/radiant-plugin-sqlite
bun link @codesordinatestudio/radiant-plugin-redis-db
bun link @codesordinatestudio/radiant-plugin-mongodb
bun link @codesordinatestudio/radiant-plugin-postgres
bun link @codesordinatestudio/radiant-plugin-nodemailer
bun link @codesordinatestudio/radiant-plugin-resend
```

> **Note:** To install them as normal dependencies later, replace `bun link` with `bun add`.
